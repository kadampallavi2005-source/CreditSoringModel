# Project Report: Credit Scoring Model

## Objective
To develop a machine learning model to predict creditworthiness.

## Methodology
1. **Data Preprocessing**: Handled missing values, encoded categorical features.
2. **Modeling**: Trained Logistic Regression, Decision Tree, and Random Forest.
3. **Evaluation**: Evaluated using Accuracy, Precision, Recall, F1 Score, and ROC-AUC.

## Results
Random Forest achieved the best performance.
